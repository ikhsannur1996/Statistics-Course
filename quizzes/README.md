# Quizzes

This folder contains quizzes for each chapter of the Statistics Course.

## Purpose

Quizzes help learners check their conceptual understanding after completing each lesson.

## Quiz Format

Each chapter quiz should include:

- Multiple-choice questions
- True or false questions
- Short answer questions
- Interpretation questions
- Code output questions when relevant

## Standard Structure

```text
quizzes/chapterXX/
└── quiz.md
```

## Guidelines

A good quiz should:

- Test understanding, not memorization only.
- Include clear answer options.
- Avoid ambiguous wording.
- Match the learning objectives of the chapter.
- Include an answer key in the corresponding solutions folder.
