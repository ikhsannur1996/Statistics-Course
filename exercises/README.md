# Exercises

This folder contains practice exercises for the Statistics Course.

## Exercise Levels

Exercises are organized into three levels:

1. Beginner — checks core understanding.
2. Intermediate — applies concepts using Python and data.
3. Advanced — requires deeper reasoning and analysis.

## Standard Exercise Structure

Each chapter should eventually include:

- 10 beginner exercises
- 10 intermediate exercises
- 10 advanced exercises
- 1 coding challenge
- 1 mini project task

## File Organization

```text
exercises/chapterXX/
├── beginner.md
├── intermediate.md
├── advanced.md
└── coding_challenge.md
```

## Goal

The full repository aims to include more than 1,000 practice questions and 100 coding challenges.
