# Roadmap

This roadmap defines the long-term development plan for Statistics Course.

Detailed pull request sequencing is tracked in [MILESTONES.md](MILESTONES.md). Each pull request should complete one milestone and preserve repository consistency.

## Phase 1 — Repository Foundation

- Create core repository documentation.
- Define project structure.
- Add requirements and environment setup.
- Add syllabus and roadmap.
- Add contribution guidelines.

## Phase 2 — Foundation Chapters

Chapters 1 to 10 will cover statistics foundations, Python, NumPy, Pandas, visualization, exploratory data analysis, and descriptive statistics.

## Phase 3 — Probability and Distributions

Chapters 11 to 25 will cover probability theory, counting principles, conditional probability, Bayes' theorem, random variables, expected value, and probability distributions.

## Phase 4 — Statistical Inference

Chapters 26 to 34 will cover sampling, the central limit theorem, confidence intervals, hypothesis testing, z-tests, t-tests, chi-square tests, and ANOVA.

## Phase 5 — Regression and Machine Learning Statistics

Chapters 35 to 45 will cover regression, diagnostics, bias and variance, feature engineering, cross validation, classification metrics, ROC, AUC, precision, recall, and F1 score.

## Phase 6 — Advanced Statistics and Capstone

Chapters 46 to 50 will cover bootstrap, Monte Carlo simulation, Bayesian statistics, time series statistics, and a final capstone project.

## Phase 7 — Polish and Release

- Add diagrams and visual assets.
- Add notebooks and solutions.
- Add datasets and data dictionaries.
- Add GitHub Actions.
- Prepare version 1.0 release.
