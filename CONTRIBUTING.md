# Contributing

Thank you for your interest in contributing to Statistics Course.

This project aims to become a high-quality open-source statistics curriculum for learners, analysts, data scientists, and educators.

## Ways to Contribute

You can contribute by:

- Improving lessons
- Adding examples
- Fixing typos
- Creating notebooks
- Adding exercises
- Adding quizzes
- Adding solutions
- Suggesting datasets
- Creating diagrams
- Improving documentation

## Content Style

All educational content should be:

- Clear and beginner-friendly
- Technically accurate
- Practical and example-driven
- Written in GitHub-flavored Markdown
- Consistent with the chapter structure

## Chapter Structure

Each chapter should include:

- Learning objectives
- Prerequisites
- Theory
- Visual explanation
- Mathematics
- Python code
- Examples
- Practice exercises
- Coding challenge
- Quiz
- Interview questions
- Mini project
- Summary
- Cheat sheet
- References

## Python Style

Python examples should:

- Use clear variable names
- Include comments where helpful
- Avoid unnecessary complexity
- Prefer reproducible examples
- Use common libraries such as NumPy, Pandas, Matplotlib, SciPy, Statsmodels, and Scikit-learn

## Pull Request Process

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Test code examples when applicable.
5. Submit a pull request with a clear explanation.

## Documentation Guidelines

- Use descriptive headings.
- Keep paragraphs readable.
- Use tables when they improve clarity.
- Use LaTeX notation for mathematical formulas when appropriate.
- Link related chapters when useful.

## Code of Conduct

All contributors are expected to follow the Code of Conduct.
