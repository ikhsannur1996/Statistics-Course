# Notebooks

This folder contains Jupyter notebooks for the Statistics Course.

## Purpose

Notebooks provide interactive, executable versions of the course lessons. They are designed to help learners practice statistics using Python.

## Notebook Structure

Each chapter should eventually include at least two notebooks:

```text
notebooks/chapterXX/
├── chapterXX_lesson.ipynb
└── chapterXX_solutions.ipynb
```

## Notebook Guidelines

Each notebook should:

- Run from top to bottom without errors.
- Include explanatory Markdown cells.
- Use clear Python code.
- Use relative paths for datasets.
- Avoid unnecessary output noise.
- Match the corresponding chapter content.

## Planned Notebook Coverage

The full course aims to include more than 100 notebooks covering statistics, data science, machine learning statistics, and business intelligence use cases.
