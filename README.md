# Statistics Course

A complete open-source Statistics Course built with Markdown lessons, Python examples, Jupyter notebooks, exercises, quizzes, mini projects, datasets, and visual assets.

## Goal

This repository aims to provide a professional statistics curriculum comparable to an online learning platform or university textbook.

## Repository Structure

```text
Statistics-Course/
├── README.md
├── requirements.txt
├── LICENSE
├── assets/
├── datasets/
├── notebooks/
├── exercises/
├── quizzes/
├── projects/
└── course/
```

## Course Outline

### Part I — Foundation

1. Introduction to Statistics
2. Python for Statistics
3. NumPy Fundamentals
4. Pandas Fundamentals
5. Data Visualization
6. Exploratory Data Analysis

### Future Parts

Additional chapters will cover descriptive statistics, probability, distributions, sampling, hypothesis testing, regression, ANOVA, machine learning statistics, and advanced topics.

## Current Chapters

- [Chapter 01 — Introduction to Statistics](course/01-introduction-to-statistics.md)

## How to Use This Course

Start from the `course/` folder and read each chapter in sequence. Use the notebooks, exercises, quizzes, and projects to practice the concepts.

Install dependencies with:

```bash
pip install -r requirements.txt
```
