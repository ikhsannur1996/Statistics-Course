# Statistics Course

An open-source, university-level statistics course built with Jupyter Notebooks.

## Course Status

This repository is being built one notebook at a time. Each notebook is designed for self-paced learning and can be executed from top to bottom.

## Getting Started

Create an environment with Python 3.12 or newer, then install the course dependencies:

```bash
pip install -r requirements.txt
```

Launch JupyterLab:

```bash
jupyter lab
```

Open the notebooks from the `notebooks/` directory and run each notebook from top to bottom.
