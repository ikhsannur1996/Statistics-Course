# Projects

This folder contains mini projects and capstone projects for the Statistics Course.

## Project Categories

Projects are organized by difficulty:

```text
projects/
├── beginner/
├── intermediate/
├── advanced/
└── capstone/
```

## Project Structure

Each project should include:

- Business problem
- Dataset description
- Exploratory data analysis
- Statistical analysis
- Visualization
- Modeling when appropriate
- Final report
- Interview questions

## Goal

The project section is designed to help learners build portfolio-ready statistics and data science projects.
