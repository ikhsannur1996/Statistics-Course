# Datasets

This folder contains datasets used throughout the Statistics Course.

## Dataset Principles

Datasets should be:

- Publicly available or clearly licensed.
- Suitable for educational use.
- Small enough for learners to download and run locally.
- Documented with source, variables, and intended use.

## Dataset Documentation Standard

Each dataset should include:

- Dataset name
- Source
- License
- Number of rows
- Number of columns
- Variable dictionary
- Example use cases
- Related chapters
